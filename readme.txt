Add Georgian Keyboard Support to Post, Page and Comment forms fields

== Description ==

<p>GeoKBD is a virtual keyboard that is intended to simplify support for Georgian Writing. Plugin requires zero configuration, all user needs is to upload GeoKBD plugin to Wordpress' plugins directory and activate it through `Plugins` page.</p>

<p>
	Additionally plugin adds corresponding box to appropriate administration pages for easily enabling/disabling Georgian Keyboard.
</p>

== Installation ==

1. Download and extract zip archive.
2. Upload "plugins" folder to /[WP_ROOT]/wp-content/plugins directory.
3. Activate GeoKBD plugin through `Plugins` page.
